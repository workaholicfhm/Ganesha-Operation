3.1 Penjelasan

praktik litas bidang informatika adalah penerapan berpikir komputasional untuk menyelesaikan berbagai permasalahan di kehidupan sehari-hari
seperti:
otonatisasi rumah, transportasi, pertanian dan lain-lain

selalu menerapkan IoT (internet of things) -> konsep dimana suatu benda atau objek ditanamkan teknologi seperti sensor dan software dengan tujuan untuk berkomunikasi, mengendalikan,
menghubungkan dan bertukar data melalui perangkat lain melalui jaringan internt 

elemen:
a. sensor -> mendeteksi input dari lingkungan sekitar
b. perangkat keras -> mellihat data sensor, mengambil keputusan untuk respon data, mengikuti aturan
c. perangkat lunak -> mengelola pengoperasian perangkat , mengelola instruksi dan meneruskannya
d. koneksi internet -> mengirim dan menerima data, intruksi dari perangkat lain

microcontroller -> perangkat keras yang digunakan untuk mengelola sensor dan di pasang perangkat lunak

Arduino uno -> papan elektronik yang digunakan untuk membuat perangkat digital, yang dapat mengendalikan perangkat lain secara interaktif

bagian dalam Arduino uno:
1. pin digital -> media input/output
2. pin analog
3. pin power -> memberikan tegangan ke perangkat elektronik lain
4. soket power -> memberikan daya ke Arduino uno
5. port usb -> upload program dari pc ke Arduino uno
6. tombol reset -> reset program Arduino

komponen yang dapat diterapkan ke Arduino -> LED, Breadboard, kabel jumper male to male, software pictoblox IDE

Led -> Light-Emiting Diode pereangkat elektronik yang merubah listrik menjadi cahaya, memiliki 2 pin, pendek untuk memberikan tegangan, panjang untuk mengeluarkan cahaya

breadboard -> papan project, papan yang digunakan untuk membuat rangkaian elektronik yang bersifat sementara, memiliki total 400 lubang yg terdiri dari warna merah merah (jalur positif yang terhubung ke pin power) dan biru (jalur negatif untuk disambungkan ke modul elektronik) jalur tengah digunakan untuk menyimpan modul elektronik, seprti sensor dll

kabel jumper male to male -> untuk menghubungkan komponen elektrinik ke Arduino uno

pictoblox ide -> software untuk menyusun program yang akan diunggah ke Arduino
