1.1.1 Analisis Data

Analisis data adalah proses pengumpulan dan pengolahan data yang akan digunakan sebagai pedoman dalam penyelesaian masalah.

Data yang sudah dikumpulkan akan diolah dalam bentuk tekstual atau visual agar mudah dimengerti.

IDE Google Colab → bahasa Python.

Python adalah bahasa pemrograman tingkat tinggi yang paling populer:

  • Pengembangan WEB
  • Aplikasi atau Software
  • Analisis Data

Web scraping adalah ekstraksi terhadap sejumlah data yang terdapat pada web tertentu.

A. Pengenalan Perkakas Analisis Data

1. Google Colaboratory

Disediakan oleh Google secara online, jadi tidak perlu melakukan instalasi.

Tekan tombol run cell (segitiga bulat) untuk melakukan running program.

2. Python

a. Cetak Data (print)


print("1 + 3")

Hasil = 1 + 3

print(1 + 3)

Hasil = 4

print("1" + "3")

Hasil = 13

Tanda petik digunakan untuk mencetak string.

String adalah teks atau kumpulan karakter.

Bisa menggunakan petik (') dan petik ganda (").

b. Pemberian Nilai Data (Assignment)

Assignment adalah pengisian atau pemberian nilai data.

Caranya adalah dengan menyediakan sebuah variabel dan diisi dengan suatu nilai menggunakan (=).


andi = 10
bila = 10
total = andi + bila

total -> hasil = 20

c. Banyak nilai untuk satu variabel (array)

Array menyimpan banyak data untuk satu variabel dengan memberi ([ ]).


data = [1, 2, 3, 4, 5, 6]
print(data)
print(data[1])

Hasil -> [1, 2, 3, 4, 5, 6]
Hasil -> 2

Array selalu dimulai dari 0.


data[2] = 10
print(data)

Hasil -> [1, 2, 10, 4, 5, 6]

.append untuk menambahkan nilai ke dalam array.


data.append(9)
print(data)

Hasil -> [1, 2, 10, 4, 5, 6, 9]

d. Penelusuran Data / Pengulangan (Loop: For)


indeks = 0
for elemen in data:
    print("elemen ke", indeks, "=", elemen)
    indeks = indeks + 1

Hasil -> Elemen ke 0 = 1
         Elemen ke 1 = 2
         Dst...

Perintah looping lain → while.

e. Pustaka Kode (Library import)

Library atau pustaka adalah koleksi program dan paket yang tersedia untuk berbagai penggunaan, contoh: pandas.

Pandas adalah library di Python yang dipakai untuk mengolah data dengan mudah seperti Excel, tetapi dijalankan dengan code.


import pandas as pd

siswa = ["Ari", "Ira", "Andi", ...]
Nilai = pd.DataFrame({
    "Siswa": siswa,
    "Nilai": nilai
})

Nilai
